# FTEL-Embedded-Project

`FTEL-Embedded-Project` là repository dùng để lưu các bài thực hành nhỏ theo hướng command-line và embedded. Phần đang có trong repo này là `mini-practice/cmd-personal`, một chương trình demo kiểu personal command manager viết bằng C, chạy trực tiếp trên Terminal để người dùng quét project, xem danh sách project có `Makefile`, rồi build, run, clean hoặc rebuild bằng các lệnh đơn giản.

## Thư mục `mini-practice/cmd-personal`

`mini-practice/cmd-personal` chứa một mini shell tương tác viết bằng C. Nó cho phép bạn quét project, build, chạy, clean và rebuild các thư mục có `Makefile` ngay trong terminal.
